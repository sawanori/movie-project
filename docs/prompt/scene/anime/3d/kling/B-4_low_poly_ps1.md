## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見
- 背景の詳細
- テクスチャ、色

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- レトロゲーム的な動き
```
