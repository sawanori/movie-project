# Kling AI 3Dスタイル B-3: ピクサー/ディズニースタイル

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | ピクサー/ディズニー風3Dアニメーション |
| 推奨語数 | 10〜25単語 |
| 推奨モデル | Kling 2.6 / Kling 2.6 Pro |

---

## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見
- 背景・環境の詳細
- 照明

✅ 書くべきこと:
- 動き・アクション（感情豊かに）
- カメラワーク
- 表現力豊かなアニメーション
```

---

## スタイルキーワード

```
Pixar animation style, expressive character animation, warm cinematic feel
```

---

## TEXT PROMPT テンプレート

```
SINGLE IMAGE RULE (do not remove):
Preserve exact character design and environment from source image.
Do not change appearance, colors, or background.

CLIP SPECIFIC (edit only this block):
Action: [EXPRESSIVE MOVEMENT - e.g., big smile, surprised reaction, joyful jump]
Camera: [CAMERA - e.g., push-in for emotion, storytelling angle]
Animation: [PIXAR MOTION - squash and stretch, anticipation, follow-through]

Pixar animation style, expressive smooth animation.
```

---

## NEGATIVE PROMPT

```
photorealistic, harsh lighting, dark and gritty, background change,
costume change, stiff animation
```

---

## 実例集

### 例1：喜びの表現

```
Action: breaks into big smile, eyes brighten with joy
Camera: slow push-in on face
Animation: squash and stretch on expression, natural follow-through

Pixar animation style, expressive smooth animation.
```

### 例2：驚きのリアクション

```
Action: reacts in surprise, leans back, eyes widen
Camera: quick push-in for impact
Animation: exaggerated anticipation, comedic timing

Pixar animation style, expressive smooth animation.
```
