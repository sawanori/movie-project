## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見・装備の詳細
- 背景・環境の詳細
- 照明セットアップ

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- ゲームシネマティック的な演出
```
