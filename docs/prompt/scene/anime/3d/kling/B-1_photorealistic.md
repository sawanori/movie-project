# Kling AI 3Dスタイル B-1: フォトリアリスティック

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | フォトリアリスティック3DCG |
| 推奨語数 | 10〜25単語 |
| 推奨モデル | Kling 2.6 Pro（高品質推奨） |

---

## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見（肌、服、髪の詳細）
- 背景・環境の詳細
- 照明セットアップ

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- 自然な動き要素
```

---

## スタイルキーワード

```
photorealistic, cinema quality, natural movement
```

---

## TEXT PROMPT テンプレート

```
SINGLE IMAGE RULE (do not remove):
Preserve exact appearance and environment from source image.
Do not change skin tones, clothing, or background.

CLIP SPECIFIC (edit only this block):
Action: [NATURAL MOVEMENT - e.g., turns head, smiles subtly, breathes]
Camera: [CAMERA - e.g., slow push-in, static, subtle drift]
Natural motion: [REALISTIC DETAILS - hair strands move, fabric settles, natural blink]

Photorealistic, cinema quality, smooth motion.
```

---

## NEGATIVE PROMPT

```
cartoon style, anime style, plastic look, background change,
costume change, unnatural skin, dead eyes
```

---

## 実例集

### 例1：ポートレート

```
Action: turns head slowly, slight smile forms
Camera: slow push-in
Natural motion: hair strands move, natural blink, subtle breathing

Photorealistic, cinema quality, smooth motion.
```

### 例2：動き出す

```
Action: takes a step forward, weight shifts naturally
Camera: tracking shot
Natural motion: clothing moves with body, hair bounces

Photorealistic, cinema quality, smooth motion.
```
