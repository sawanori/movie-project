# Kling AI 2Dアニメスタイル A-2: ジブリ風

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | スタジオジブリ風（手描き感のある温かみ） |
| 推奨語数 | 10〜25単語 |
| 推奨モデル | Kling 2.6 / Kling 2.6 Pro |

---

## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見
- 背景・環境の詳細
- 色、照明

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- 自然で穏やかな動き
- スタイルキーワード
```

---

## スタイルキーワード

```
Ghibli inspired style, hand-drawn animation, gentle naturalistic movement
```

---

## TEXT PROMPT テンプレート

```
SINGLE IMAGE RULE (do not remove):
Preserve exact character design and background from source image.
Do not change appearance, colors, or environment.

CLIP SPECIFIC (edit only this block):
Action: [GENTLE MOVEMENT - e.g., walks slowly, looks up, reaches out]
Camera: [CAMERA - e.g., static shot, slow pan]
Animation: [NATURAL MOTION - wind in hair, grass sways, fabric settles]

Ghibli inspired style, gentle naturalistic movement.
```

---

## NEGATIVE PROMPT

```
photorealistic, 3D render, harsh lighting, background change,
costume change, fast action, dramatic effects
```

## 実例集

### 例1：風を感じる

```
Action: closes eyes peacefully, tilts head up slightly
Camera: static shot
Animation: hair and clothing sway gently in breeze

Ghibli inspired style, gentle naturalistic movement.
```

### 例2：歩く

```
Action: walks forward slowly with calm expression
Camera: tracking shot
Animation: hair bounces softly, dress flows

Ghibli inspired style, gentle naturalistic movement.
```
