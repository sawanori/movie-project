# Kling AI 2Dアニメスタイル A-4: フラットデザイン

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | モダンフラットデザイン / ベクターアニメーション風 |
| 推奨語数 | 10〜25単語 |
| 推奨モデル | Kling 2.6 / Kling 2.6 Pro |

---

## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見
- 背景の詳細
- 色、形状

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- グラフィカルな動き
- スタイルキーワード
```

---

## スタイルキーワード

```
flat design animation, vector art style, clean graphic motion
```

---

## TEXT PROMPT テンプレート

```
SINGLE IMAGE RULE (do not remove):
Preserve exact design and background from source image.
Do not change colors, shapes, or environment.

CLIP SPECIFIC (edit only this block):
Action: [GRAPHIC MOTION - e.g., slides in, bounces, transforms]
Camera: [CAMERA - e.g., static, smooth pan]
Animation: [CLEAN MOTION - smooth tweening, shape morphing]

Flat design animation, clean graphic motion.
```

---

## NEGATIVE PROMPT

```
photorealistic, detailed textures, complex shading, background change,
color change, messy lines
```

---
