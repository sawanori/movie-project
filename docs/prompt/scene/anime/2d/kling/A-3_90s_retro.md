# Kling AI 2Dアニメスタイル A-3: 90年代レトロ

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | 90年代セルアニメーション風 |
| 推奨語数 | 10〜25単語 |
| 推奨モデル | Kling 2.6 / Kling 2.6 Pro |

---

## 重要原則：画像優先

```
⚠️ 画像にある情報はプロンプトに書かない

❌ 書いてはいけない:
- キャラクターの外見
- 背景・シーンの詳細
- 色、照明

✅ 書くべきこと:
- 動き・アクション
- カメラワーク
- レトロアニメ的な演出
- スタイルキーワード
```

---

## スタイルキーワード

```
90s anime style, retro cel animation, classic anime movement
```

---

## TEXT PROMPT テンプレート

```
SINGLE IMAGE RULE (do not remove):
Preserve exact character design and background from source image.
Do not change appearance, colors, or environment.

CLIP SPECIFIC (edit only this block):
Action: [DRAMATIC MOVEMENT - e.g., strikes pose, turns dramatically]
Camera: [CAMERA - e.g., zoom punch, dramatic angle]
Animation: [RETRO MOTION - speed lines feel, dramatic pause, held pose]

90s anime style, classic anime movement.
```

---

## NEGATIVE PROMPT

```
photorealistic, modern digital art, 4K sharp, background change,
costume change, smooth gradients
```
