# HailuoAI 2Dアニメスタイル A-2: ジブリ風

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | ジブリ風（手描きアニメ） |
| 推奨語数 | 10〜40単語 |
| 推奨モデル | MiniMax-Hailuo-02 |
| 特徴 | カメラコマンド `[command]` 構文 |

---

## スタイルキーワード

```
Ghibli style, hand-drawn animation, watercolor backgrounds, soft lighting
```

---

## TEXT PROMPT テンプレート

```
[CAMERA_COMMAND]

Action: [GENTLE MOVEMENT - e.g., looks up at sky, wind blows through hair]
Animation: [NATURAL MOTION - grass sways, clouds drift, leaves flutter]

Ghibli style, hand-drawn animation, soft lighting.
```

---

## 実例集

### 例1：風を感じるシーン

```
[Static shot]

Action: closes eyes, feels the wind
Animation: hair and clothes flutter gently, grass sways

Ghibli style, hand-drawn animation, peaceful atmosphere.
```

### 例2：歩くシーン

```
[Tracking shot]

Action: walks forward with gentle steps
Animation: dress sways, hair bounces naturally

Ghibli style, hand-drawn animation, soft movement.
```

### 例3：空を見上げる

```
[Tilt up]

Action: looks up at the sky with wonder
Animation: subtle eye movement, hair shifts

Ghibli style, hand-drawn animation, dreamlike quality.
```
