# HailuoAI 2Dアニメスタイル A-4: フラットデザイン

## 概要

| 項目 | 内容 |
|------|------|
| スタイル | フラットデザイン・ミニマル |
| 推奨語数 | 10〜40単語 |
| 推奨モデル | MiniMax-Hailuo-02 |
| 特徴 | カメラコマンド `[command]` 構文 |

---

## スタイルキーワード

```
flat design, minimal animation, simple shapes, clean lines
```

---

## TEXT PROMPT テンプレート

```
[CAMERA_COMMAND]

Action: [SIMPLE MOVEMENT - e.g., simple gesture, minimal motion]
Animation: [CLEAN MOTION - smooth transitions, simple movement]

Flat design, minimal animation, clean aesthetic.
```
